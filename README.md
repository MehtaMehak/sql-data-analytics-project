# sql-data-analytics-project
This repository contains SQL scripts showcasing various data analysis techniques, including time-based trend analysis, cumulative metrics, performance evaluation, data segmentation, and part-to-whole analysis to extract meaningful business insights
SQL Data Analytics Project
Project Overview

This repository contains a SQL-based data analytics project demonstrating various data exploration and analytical techniques used by data analysts to extract insights from relational databases. The project focuses on transforming raw data into meaningful insights using structured SQL queries and analytical methods.

The analysis covers both Exploratory Data Analysis (EDA) and Advanced Analytics techniques to understand data patterns, evaluate performance, and support data-driven decision-making.

Tools Used

SQL

Relational Database

Data Analysis Techniques

Exploratory Data Analysis (EDA)

The project begins with exploring and understanding the dataset using SQL.

Key analyses performed include:

Database Exploration – Understanding tables, schema, and data structure

Dimension Analysis – Analyzing categorical attributes and data distribution

Date Exploration – Identifying time-based patterns in the dataset

Measures Exploration – Evaluating key numerical metrics

Magnitude Analysis – Comparing the scale of key values

Ranking Analysis – Identifying Top-N performing entities

Advanced Analytics

The project also demonstrates advanced analytical techniques used in real-world data analysis:

Change Over Time Analysis – Identifying trends across different time periods

Cumulative Analysis – Tracking cumulative performance over time

Performance Analysis – Comparing performance across different categories

Part-to-Whole Analysis – Understanding contribution of components to totals

Data Segmentation – Grouping data into meaningful segments

Reporting Queries – Creating structured queries for analytical reporting

SQL Concepts Used

Aggregations (SUM, AVG, COUNT)

GROUP BY and HAVING

Window Functions

Ranking Functions

Joins

CASE Statements

Conclusion

This project demonstrates how SQL can be used for data exploration, performance analysis, and advanced analytics to generate business insights from structured data.
