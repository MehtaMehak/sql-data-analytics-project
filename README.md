# SQL Data Analytics Project

## 📌 Overview
This project demonstrates how SQL can be used to perform data analysis on relational databases, transforming raw data into meaningful business insights.

It covers both **Exploratory Data Analysis (EDA)** and **Advanced Analytics techniques** used by data analysts.

---

## 🛠 Tools Used
- SQL  
- Relational Database  

---

## 🔍 Exploratory Data Analysis (EDA)
- Database exploration (tables, schema, structure)
- Dimension analysis (categorical data)
- Date analysis (time-based trends)
- Measures analysis (key metrics)
- Ranking analysis (Top-N entities)

---

## 📊 Advanced Analytics
- Change over time analysis  
- Cumulative analysis  
- Performance comparison  
- Part-to-whole analysis  
- Data segmentation  
- Reporting queries  

---

## ⚙️ SQL Concepts Used
- Aggregations (SUM, AVG, COUNT)  
- GROUP BY, HAVING  
- Joins  
- Window functions  
- Ranking functions  
- CASE statements  
- Date functions  

---

## 🎯 Key Outcome
Built a structured set of SQL queries to analyze business data, identify trends, and generate actionable insights for decision-making.
